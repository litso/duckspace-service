duckspace-service
=================

API and Web

Set up dev

    $ bundle

Boot local

    $ rackup

Set up heroku

    $ heroku git:remote -a duckspace

Push develop to heroku

    $ git push heroku develop:master

Push master to heroku

    $ git push heroku master
